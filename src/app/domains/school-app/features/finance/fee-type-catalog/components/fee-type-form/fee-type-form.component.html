Tarification (optionnel)

Si renseigné, ce type de frais peut être facturé automatiquement quand un élève souscrit au service correspondant à l'inscription (Cantine, Transport...) — le code technique doit alors correspondre exactement au code du service côté inscription. Laisser vide pour un type de frais saisi manuellement (Inscription, Scolarité, frais ponctuels).

Scolarité : chaque option correspond à un niveau scolaire — le tarif s'applique automatiquement selon le niveau souhaité par la famille.

Chaque option correspond à un choix possible pour ce service (ex. zone de transport, formule de cantine) — le code doit correspondre exactement à l'option choisie côté inscription.
