# Ton de Voix Feewi

## 1. Pour le Parent (Rassurant)
- **Exemple Bouton :** "Soumettre mon dossier"
- **Exemple Erreur :** "Veuillez vérifier le format du numéro de téléphone."
- **Adjectifs :** Bienveillant, clair, patient.

## 2. Pour le Secrétariat (Efficace)
- **Exemple Bouton :** "Valider le dossier"
- **Exemple Erreur :** "Champ requis manquant."
- **Adjectifs :** Professionnel, direct, neutre.

## 3. Anti-Jargon (Mapping)
- `UUID` / `ID` -> "Référence" ou "Numéro"
- `Bundle` -> "Lot" ou "Dossier"
- `Tenant` -> "École" ou "Établissement"
- `Submit` -> "Envoyer" ou "Soumettre"
