# Profils Utilisateurs Feewi

## 1. Le Parent (Portail Inscription)
- **Contexte :** Usage unique/annuel, souvent sur mobile (50-60%).
- **Besoin :** Vitesse, simplicité, réassurance.
- **UX :** Parcours guidé, gros boutons, pas de jargon, indicateur de progression.

## 2. Le Secrétariat (App Admissions)
- **Contexte :** Usage quotidien, desktop exclusif, haute pression.
- **Besoin :** Efficacité, vitesse de traitement, densité d'information.
- **UX :** Raccourcis, listes denses, filtres rapides, bulk actions.

## 3. La Direction (Dashboard)
- **Contexte :** Usage décisionnel, sessions courtes.
- **Besoin :** Vue d'ensemble, KPIs clairs.
- **UX :** Tableaux de bord épurés, actions de validation proéminentes.
