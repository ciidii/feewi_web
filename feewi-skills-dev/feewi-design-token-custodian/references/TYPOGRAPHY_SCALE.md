# Échelle Typographique Feewi

## 1. Polices
- **Display / Brand :** Lexend (700-800)
- **Interface :** Inter (400-700)
- **Code :** JetBrains Mono

## 2. Échelle par Fonction
| Fonction | Police | Poids | Taille | Line-height |
|---|---|---|---|---|
| **Display LG** | Lexend | 800 | 48px | 1.1 |
| **Display MD** | Lexend | 700 | 36px | 1.1 |
| **Display SM** | Lexend | 700 | 28px | 1.2 |
| **Heading LG** | Inter | 700 | 24px | 1.3 |
| **Heading MD** | Inter | 700 | 20px | 1.4 |
| **Heading SM** | Inter | 600 | 18px | 1.4 |
| **Title LG** | Inter | 600 | 16px | 1.5 |
| **Title MD** | Inter | 600 | 14px | 1.5 |
| **Title SM** | Inter | 700 | 12px | 1.4 |
| **Body LG** | Inter | 400 | 16px | 1.6 |
| **Body MD** | Inter | 400 | 14px | 1.6 |
| **Body SM** | Inter | 400 | 13px | 1.5 |
| **Caption** | Inter | 500 | 11px | 1.4 |
| **Label** | Inter | 700 | 10px | 1.2 |

## 3. Règles
- Pas plus de 2 tailles sur une même carte.
- Jamais de texte < 11px.
- Labels ALL CAPS réservés aux métadonnées.
